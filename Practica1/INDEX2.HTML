Este es el nuevo titulo de mi nueva pagina que esta bien chida

El dia de hoy aprendi a crear un archivo con extension .html, en el cual puedo tener una pagina web, personalizandola etc, para crear el archivo primero tenemos que crear una carpeta en el directorio de trabajo que lleve por nombre "html" despues creamos un nuevo archivo al cual nombramos index entonces insertamos el comando Shift + 1 + Enter para asegurarnos o confirmar que estamos usando el HTML5, despues agregamos un nombre a la pagina en la seccion de "title" y por ultimo, en la seccion de "body" finalizamos añadiendo etiquetas, para eso tecleamos "menor que + ! + -- " y agregamos la etiqueta, siendo el comando "h" para agregar titulos y subtitulos, "p" para agregar parrafos Ademas podemos usar "lorem" para agregar un parrafo o texto aleatorio, tambien podemos anexar un numero para delimitar las palabras del texto aleatorio o multiplicarlo con "*" para que tenga un numero estipulado de palabras tambien podemos usar saltos de linea con el comando "br" y añadir textos personalizados con el comando "pre" Por ultimo aprendi a añadir enlaces con el comando "A" donde despues del href= insertamos el enlace al que queremos que rediriga nuestra pagina web añadiendo una pequeña instruccion, por ejemplo "Ir a Facebook"
